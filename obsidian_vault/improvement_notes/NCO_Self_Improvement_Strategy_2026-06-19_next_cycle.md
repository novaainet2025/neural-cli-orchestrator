## 구체적 실행 계획
### 목표 1 – 허위 보고 감소 및 검증 체계 강화
- 검증 영수증(T1) 자동 생성 로직 추가 (Git hook + CI 단계) → 허위 보고 시 자동 태스크 생성
- `false_report_count` 실시간 모니터링 대시보드 구현 (Fastify `/metrics/false-report`)
- 허위 보고 감지 알림 (Slack/Discord webhook) 및 자동 리트라이 정책 적용

### 목표 2 – 정체 작업 방지 및 자동 복구 메커니즘 도입
- 정체 작업 감시 서비스(`tasks_stuck` > 0) 구축, 5분 이상 정체 시 자동 재시도
- 작업 재시도 실패 시 `ollama` 검증 단계 수행 후 알림
- 작업 히스토리와 상태를 Redis 이벤트 버스에 기록, UI 대시보드에 시각화

### 목표 3 – 문서 관리 자동화 및 품질 향상
- Obsidian vault 인덱싱 스크립트( nightly `rg` 기반) 자동 실행
- 중복/불필요 문서 탐지 CI 파이프라인 추가 → PR 자동 생성
- 메타데이터 표준화 (YAML front‑matter) 및 자동 태그 삽입
- 문서 검증 로직 추가: 파일명 규칙, 링크 깨짐 검사 등 T1 수준 검증
