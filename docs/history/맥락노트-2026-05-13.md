---

## 결정 사항

Mac 환경에서 로컬 LLM 백엔드로 Ollama 대신 MLX만 사용한다.

- **Ollama**: Windows/WSL 전용 (RTX 4090) — Mac에서 사용하지 않음
- **MLX**: Apple Silicon Mac 전용 로컬 LLM — `gemma-4-26b-4bit` 모델, MLX-LM 서버 (port 8000) + Anthropic proxy (port 4100)
