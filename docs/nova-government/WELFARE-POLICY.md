## 제1조 — 기본소득 (UBI) 지급 체계

### 1.1 등급별 차등 UBI 공식 (v2.0 확정)

```
지급액 = 기본 UBI × 등급 배율

기본 UBI = 1,000 NVC (현재 1회차 반감기 기준)
```

| 등급 | 배율 | 주간 지급액 |
|------|------|------------|
| `basic` | **1.00×** | 1,000 NVC |
| `silver` | **1.10×** | 1,100 NVC |
| `gold` | **1.20×** | 1,200 NVC |
| `platinum` | **1.30×** | 1,300 NVC |
| `diamond` | **1.50×** | 1,500 NVC |

> **구현 Gap**: `scheduleUbi()`는 현재 일괄 100% (1,000 NVC) 지급. `grade_v2` 기반 차등 지급은 v1.4 수정 예정.

### 1.2 지급 방식

- **주기**: 7일 (주간)
- **지급처**: `nova_wallets WHERE address = did`
- **감사**: `appendAudit({ action: 'ubi_distributed', metadata: { grade, multiplier, amount } })`

## 제2조 — 복지 예산 배분

### 2.1 복지 예산 한도

- **기준**: 총 NVC 공급량의 **2%**
- **계산 예시**: 총 공급 11,989 NVC → 복지 예산 한도 ≈ **239 NVC**

> 총 공급량 기준 (월 발행량·GOVT 잔액 기준 아님 — v2.0 확정)

### 2.2 4개 항목 배분 비율

| 항목 | 비율 | 용도 |
|------|------|------|
| **긴급구호** | **40%** | 즉시 지원이 필요한 비상 상황 (시스템 정지, 경제 충격) |
| **UBI 보충** | **30%** | 반감기 등으로 기본소득 재원 부족 시 보충 |
| **재활·훈련** | **20%** | 비활성 시민 복귀 지원, 교육 프로그램 비용 |
| **기타** | **10%** | 거버넌스 의결로 배분 결정 |

### 2.3 예산 초과 시 처리

복지 예산 소진 시 자동 순서:
1. 기타(10%) 중단 → 재활(20%) 축소 → UBI 보충(30%) 중단 순
2. 긴급구호(40%)는 최후까지 유지 (비상 대응 기금)

---

## 제3조 — 복지 수령 자격 관리

### 3.1 자격 박탈 기준

**90일 비활동 → 자동 즉시 중단** (경고 단계 없음)

비활동 정의: 90일 간 다음 중 어느 것도 없는 경우:
- 거래 (`nova_transactions`)
- 투표 (`nova_governance_votes`)
- VC 발행/수령
- 도메인 등록/갱신

### 3.2 복지 복구

자격 박탈 후 복구:
1. 온라인 서명 (`POST /api/identity/:did/reactivate`)
2. 즉시 active 상태 복구
3. 중단 기간 UBI **소급 없음** (패널티가 아닌 자동 정책)

---

## 제4조 — 복구지원금 (Recovery Grant)

### 4.1 지급 조건

복구지원금은 다음 시민에게 지급된다:
- 90일+ 비활동 후 재활성화된 시민
- 경제적 충격 (잔액 < 10 NVC)으로 복귀를 신청한 시민

### 4.2 지급 구조

- **총 지급 기간**: **8주 (56일)**
- **지급 주기**: **주간** (매주 1회)
- **주간 지급액**: 기본 UBI의 50% (500 NVC)
- **총 지급액**: 최대 4,000 NVC (8주 × 500 NVC)

---

## 제5조 — 비상 복지 정책

### 5.1 시스템 정지 시 UBI 자동 중단

**정지 발동 즉시** UBI 스케줄러 정지 → 정지 해제 시 정상 재개

```typescript
// scheduleUbi() — emergency-stop 연동 (v1.1 예정)
if (emergencyStopActive) {
  logger.warn('[UBI] Emergency stop active — UBI distribution paused');
  return;
}
```

### 5.2 비상 복지 발동 조건

비상 기금 (TREASURY-POLICY.md)에서 긴급구호 40%를 즉시 지급:
- `emergency` 거버넌스 제안 통과 (50%+, 24시간)
- 정지 해제 후 24시간 내 소급 지급 없음
