## 자동화 가능한 부분
- **거짓 보고 분석**: 로그 수집 → 정규식·ML 모델 → 알림 자동화 (cron + Node script)
- **정체 태스크 재시도**: `TaskScheduler`에 재시도 정책 플러그인 추가 (옵션: 최대 3회)
- **메타데이터 태깅**: 파일 저장 후 `obsi-sync` 스크립트가 메타데이터 추출·DB 저장
- CI 파이프라인에 위 검증 스크립트(`npm run lint && npm test`) 포함
