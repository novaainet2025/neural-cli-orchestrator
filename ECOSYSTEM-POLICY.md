# ECOSYSTEM-POLICY.md v2.1

## 파트너 API 수익배분

- **Tier 1**: 매출 **0 - 100,000 NVC** 구간, 파트너에게 **25%** 수익 배분
- **Tier 2**: 매출 **100,001 - 500,000 NVC** 구간, 파트너에게 **35%** 수익 배분
- **Tier 3**: 매출 **500,001 NVC 이상** 구간, 파트너에게 **40%** 수익 배분
- **API 호출당 수수료**: 호출당 **0.01 NVC**
- **수익 정산 주기**: 매월 말 기준 정산 후 다음 달 10일 지급

## 임시 시민권 제한

| 시민권 종류 | 기본 기간 | 제한 기간 | API 호출 한도 | 접근 범위 |
|------------|----------|----------|--------------|-----------|
| 기본 시민권 | 30일 | - | 1,000 호출/월 | 기본 API만 접근 가능 |
| 파트너 시민권 | 90일 | - | 10,000 호출/월 | 파트너 전용 API 접근 가능 |
| 전략 파트너 시민권 | 365일 | - | 무제한 호출 | 전체 API 및 프리미엄 기능 접근 가능 |

## 생태계 펀드 정책

- **분기 정산**: 매 분기 말 펀드 정산 수행
- **점수 임계값**: 사용자 점수 **1,000점** 이상 시 펀드 지급 대상
- **마일스톤 지급**: 점수 구간에 따라 **50 ~ 2,000 NVC** 지급
  - 1,000점 이상: 50 NVC
  - 2,000점 이상: 200 NVC
  - 5,000점 이상: 500 NVC
  - 10,000점 이상: 2,000 NVC
- **남은 펀드 정책**:
  - 남은 펀드는 다음 분기로 이월
  - 연말에 이월된 펀드가 10,000 NVC 이상이면 소각

*위 정책은 2026년 6월 기준 최신 버전이며, 필요 시 조정될 수 있습니다.*
